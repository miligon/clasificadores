# Practicas de IA (clasificadores de imagenes)

Implementación de los algoritmos KNN y teorema de bayes, para la clasificación de imágenes

Requisitos:
```
pip install numpy
```

Para ejecutar:
```
python main.py
```

Iris Data Set:
https://archive.ics.uci.edu/ml/datasets/iris
